## Commands

/FancyReferrals (version | reload | refresh)<br>
/FancyReferrals refresh - refreshes the top10 leaderboard<br>
/Referral (player | top)<br>

## Permissions
/FancyReferrals - ``fancyreferrals.admin``<br>
/Referral - ``fancyreferrals.referral``
